# Transport Order General

Web-based transport order form (PDF generator) for general and refrigerated shipments.

## Features

- Create transport orders for shipments.
- Supports general and refrigerated shipments.
- Generates PDF files ready for printing.
- User-friendly interface with dynamic form fields.
- Responsive layout for desktop and mobile.

## File Structure

- `index.html` – Main web page for creating and printing transport orders.

## Usage

1. Open `index.html` in a web browser.
2. Fill in the transport order form.
3. Click "Generate PDF" to create a printable PDF file.
